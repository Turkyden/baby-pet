# Baby Pet

A plateform designed for the information publish about pet service.

@ By Turkyden & Liuliu

## Get Starter

1) 全局安装 `@tarojs/cli` 脚手架

``` npm
npm install -g @tarojs/cli
```

2) 进入 `client` 文件夹

``` cmd
cd baby-pet/client
```

3) 安装工程依赖

``` npm
npm install
```

4) 启动小程序客户端开发

``` npm
npm run dev
```

``` npm
npm run build
```
